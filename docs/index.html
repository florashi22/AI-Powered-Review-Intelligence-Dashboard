Architecture

How it works

01 — Input

Raw review text

Unstructured customer feedback — any length, any topic, any platform.

02 — Orchestration

Claude API reasoning

Claude reads the review like a domain expert and returns a typed JSON object — not just a label.

03 — Output

Structured intelligence

Star rating, confidence, sentiment, themes, probability distribution, and actionable business advice.

Orchestration pattern
Review text  →  Claude API (orchestrator)  →  Structured JSON
                                               ├── stars: 4
                                               ├── confidence: "82%"
                                               ├── sentiment: "Mixed"
                                               ├── themes: [{label, type}, ...]
                                               ├── star_probs: {1:2, 2:5, ...}
                                               └── advice: "..."
