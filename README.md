# AI-Powered Review Intelligence Dashboard

Turn unstructured customer reviews into ratings, insights, and business decisions.

## Overview
This project builds an end-to-end machine learning system that predicts Yelp-style ratings from customer reviews, compares bag-of-words and transformer-based approaches, and provides interpretable insights through an interactive dashboard.

## Key Features
- Predict review star rating from text
- Confidence scores and probability distribution
- Model comparison (BoW vs Transformer)
- Error analysis and confusion matrix
- Business insights from customer feedback

## Tech Stack
- Python
- Streamlit
- scikit-learn
- transformers (Hugging Face)
- matplotlib
